git-私库gitolite搭建

1. 服务器创建git账号

useradd git -d /data/git #创建用户git，设置主路径为/data/git
passww git #修改密码

2. 服务器安装gitolite

git clone git://github.com/sitaramc/gitolite

#####安装
gitolite/install -to /data/git/bin/

将管理员客户端的clzmac.pub上传至主目录,必须确保~/.ssh/authorized_keys为空才能增加git的管理员key

/data/git/bin/gitolite setup -pK clzmac.pub
初始化空的 Git 版本库于 /home/git/repositories/gitolite-admin.git/
初始化空的 Git 版本库于 /home/git/repositories/testing.git/

3. 管理员管理gitolite

hexo搭建

1. 安装git

yum install git

git －version
检查安装状态

2. 安装node.js

node.js

wget https://nodejs.org/dist/v0.12.6/node-v0.12.6.tar.gz
tar -zxvf node-v0.12.6.tar.gz
cd node-v0.12.6
./configure
make && make install

node -v 检查安装状态

3. 安装hexo

npm install -g hexo-cli
